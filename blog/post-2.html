Python Basics for Absolute Beginners

🗓️ May 31, 2025

Welcome! 🎉 If you're just getting started with Python, you're in the right place. This post will walk you through the core concepts every beginner should know.

✨ 1. Why Python?

  • ✅ Easy to read and write
  • ✅ Huge community & tons of libraries
  • ✅ Used in web dev, data science, automation, AI...

📦 2. Installing Python

Download it from python.org and make sure to check “Add to PATH” when installing. Then open a terminal and type:

python --version

🔤 3. Variables & Data Types

# Strings
name = "Luna"

# Integers & floats
age = 20
height = 1.70

# Booleans
is_coding = True

🔁 4. Control Flow (if/else, loops)

if age > 18:
    print("You're an adult!")

for i in range(3):
    print("Hello", i)

🧠 5. Functions

def greet(name):
    return f"Hello, {name}!"

print(greet("Luna"))
