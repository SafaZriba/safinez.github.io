What Are Intervals in Programming? ⏱️

An interval represents a range between two values.

It usually looks like this:

[start, end]

For example:

  • [1, 3]
  • [2, 6]
  • [8, 10]

These ranges are used to represent things like time slots, events, or numeric ranges.

📦 Real-World Example

Imagine scheduling meetings during the day:

  • Meeting A → [9, 10]
  • Meeting B → [9:30, 11]
  • Meeting C → [13, 14]

The first two meetings overlap. The third one does not.

🧩 Common Interval Problems

  • Merging overlapping intervals
  • Finding conflicts in schedules
  • Booking meeting rooms
  • Finding free time

⚡ Key Idea

Most interval problems follow one powerful trick:

Sort the intervals by their starting time.

Once sorted, it becomes easy to detect overlaps or merge ranges.

💻 Example: Merge Intervals (Python)


def merge(intervals):
    intervals.sort()
    merged = [intervals[0]]

    for start, end in intervals[1:]:
        last_end = merged[-1][1]

        if start <= last_end:
            merged[-1][1] = max(last_end, end)
        else:
            merged.append([start, end])

    return merged
Intervals turn messy timelines into clean, solvable patterns.
