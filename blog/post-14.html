What Is Binary Search? 🔍

Binary Search is a fast way to find a value inside a sorted list.

📦 Real-World Example:

Imagine looking for a name in a phonebook. You don’t start from page 1.

You open it in the middle.

  • If the name comes before → go left
  • If it comes after → go right

Each step cuts the search space in half.

⚡ Why It’s Powerful:

  • Much faster than checking one by one
  • Works in O(log n) time
  • Perfect for large sorted data

🧠 Important Rule:

The list MUST be sorted.

Divide the problem in half until the answer reveals itself.

💻 Example in Python:


def binary_search(nums, target):
    left = 0
    right = len(nums) - 1

    while left <= right:
        mid = (left + right) // 2

        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

numbers = [1, 3, 5, 7, 9, 11]
print(binary_search(numbers, 7))  # Output: 3
