What Is a Trie? 🌲

A Trie is a special type of tree used to store strings efficiently.

📦 Real-World Example:

Think of autocomplete when you type in Google.

You type “app”… and it suggests:

  • apple
  • application
  • apply

All these words share the same prefix.

🧩 How It Works:

Each node represents a letter.

Words are formed by following paths from the root.

Nodes share common prefixes.

⚡ Why It’s Powerful:

  • Fast prefix searching
  • Efficient autocomplete systems
  • Saves memory by sharing prefixes
A Trie grows letter by letter.

💻 Simple Python Example:


class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end = True

# Example
trie = Trie()
trie.insert("apple")
trie.insert("app")
