What Are Graphs? 🌐

A Graph is a data structure used to represent relationships between objects.

It consists of two main parts:

  • Nodes (Vertices) → the objects
  • Edges → the connections between them

📦 Real-World Examples:

  • Social networks (people connected to friends)
  • Google Maps (cities connected by roads)
  • The internet (computers connected by networks)

🧩 Types of Graphs:

  • Undirected Graph → connections go both ways
  • Directed Graph → connections have direction
  • Weighted Graph → edges have values like distance or cost

⚡ Why Graphs Are Powerful:

  • Model complex relationships
  • Help find shortest paths
  • Used in networking, AI, and search engines
If trees describe hierarchy, graphs describe the real world.

💻 Simple Graph Example in Python:


graph = {
  "A": ["B", "C"],
  "B": ["A", "D"],
  "C": ["A", "D"],
  "D": ["B", "C"]
}

print(graph["A"])  # neighbors of A
