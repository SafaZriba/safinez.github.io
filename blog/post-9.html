If Statements in Programming 🤖

An if statement is a decision — just like in real life.

💡 Real Life Logic:

  • If it rains → take an umbrella ☔
  • If you're hungry → eat 🍕
  • If the light turns green → go 🚦

🧠 In Code, It Looks Like:

if itRains:
   takeUmbrella()

📌 Why If Statements Matter:

  • Help computers make choices
  • Add logic to your programs
  • Make software behave smartly
Computers don’t guess — they wait for a condition to be true.
