What Is Backtracking? 🔄

Backtracking is an algorithm technique used to explore all possible solutions by trying, undoing, and trying again.

📦 Real-World Example:

Imagine solving a maze.

  • You choose a path.
  • If you hit a dead end, you go back.
  • Then you try another path.

That “go back and try again” behavior is backtracking.

🧩 Where It’s Used:

  • Sudoku solvers
  • N-Queens problem
  • Generating permutations
  • Combination problems

⚡ Why It’s Powerful:

  • Explores all valid possibilities
  • Avoids unnecessary paths early
  • Works well with recursion
Try. Fail. Undo. Try again.

💻 Simple Example (Generating Subsets):


def backtrack(start, path):
    print(path)

    for i in range(start, len(nums)):
        path.append(nums[i])
        backtrack(i + 1, path)
        path.pop()  # Undo choice

nums = [1, 2, 3]
backtrack(0, [])
