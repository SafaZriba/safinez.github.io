What Is 1-D Dynamic Programming? 🧠

1-D Dynamic Programming is a technique used to solve problems by storing solutions to smaller subproblems in a single array.

Instead of solving the same problem again and again, the algorithm remembers previous results.

📦 Real-World Example

Imagine climbing a staircase.

You can take either 1 step or 2 steps at a time.

The question is: How many ways can you reach the top?

Instead of recomputing every possibility, we store the number of ways to reach each step.

🧩 Why It’s Called 1-D

The solutions are stored in a single list (array).

  • dp[0]
  • dp[1]
  • dp[2]
  • dp[3]

Each position represents the solution for that step.

⚡ Why It’s Powerful

  • Avoids repeated work
  • Makes recursive problems efficient
  • Often reduces exponential time to linear time
Dynamic Programming is just recursion with memory.

💻 Example in Python (Climbing Stairs)


def climbStairs(n):
    dp = [0] * (n + 1)

    dp[0] = 1
    dp[1] = 1

    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]

    return dp[n]

print(climbStairs(5))
