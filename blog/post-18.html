What Is a Tree in Data Structures? 🌳

A Tree is a hierarchical data structure made of nodes connected like branches.

📦 Real-World Examples:

  • Folder system on your computer
  • Organization charts
  • Family trees

🧩 Basic Terms:

  • Root → The top node
  • Parent → A node that has children
  • Child → A node connected below another
  • Leaf → A node with no children

🌿 Special Type: Binary Tree

A Binary Tree is a tree where each node has at most two children:

  • Left child
  • Right child

⚡ Why Trees Matter:

  • Efficient searching and sorting
  • Organizing hierarchical data
  • Used in databases and AI systems
Trees allow data to grow in multiple directions.

💻 Simple Python Example:


class TreeNode:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None

# Creating nodes
root = TreeNode(10)
root.left = TreeNode(5)
root.right = TreeNode(15)

print(root.value)        # 10
print(root.left.value)   # 5
print(root.right.value)  # 15
