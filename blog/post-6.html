What Are Arrays (Lists) in Python?

Arrays are collections of values ‚Äî think of them like labeled boxes that hold numbers, words, or other items.

print(numbers[0])  # Output: 1

for num in numbers:
    print(num)

Tip: Indexing in Python starts at 0!

What Are Strings?

Strings are sequences of characters. In Python, they behave a lot like arrays of letters!

greeting = "hello"
print(greeting[1])       # Output: 'e'
print(greeting + " world")  # Output: 'hello world'
print(greeting[1:4])     # Output: 'ell'
print(len(greeting))     # Output: 5

Example: Reversing a String

def reverse_string(s):
    return s[::-1]

print(reverse_string("hello"))  # Output: 'olleh'
