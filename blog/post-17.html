What Is a Stack? 🥞

A Stack is a data structure that follows the principle:

LIFO — Last In, First Out.

📦 Real-World Example:

Think of a stack of plates.

  • You add a plate on top (push)
  • You remove the top plate (pop)

You cannot remove the plate at the bottom directly.

🧩 Main Operations:

  • Push → Add an item to the top
  • Pop → Remove the top item
  • Peek → View the top item without removing it

⚡ Where Stacks Are Used:

  • Undo/Redo functionality
  • Browser history
  • Function calls (Call Stack)
  • Checking balanced parentheses
A Stack remembers the most recent action first.

💻 Example in Python:


stack = []

# Push
stack.append(10)
stack.append(20)
stack.append(30)

# Pop
print(stack.pop())  # Output: 30

# Peek
print(stack[-1])    # Output: 20
